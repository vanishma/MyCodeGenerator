# MyCodeGenerator

生成 curd 代码

## 模板能使用的数据

| 名称            | 数据类型 | 说明        |
|----------------|---------|-----------|
| importPackages | Array   | 需要导入的包    |
| lombok         | boolean | lombok 模型 |
| tableComment   | String  | 表名        |
| fileName       | String  | 文件名       |
| typeName       | Array   | 数据类型集合    |
| name           | Array   | 字段名       |
| classPath| String | 包路径|
